# Garden Planner App Icon

To generate the app icons, you need to:

1. Create a 1024x1024px PNG image named `garden_icon.png` with a garden/plant theme (e.g., a sprouting plant, garden bed, or vegetable icon in green tones)

2. Create a 1024x1024px PNG image named `garden_icon_foreground.png` for Android adaptive icons (foreground layer)

3. Run: `flutter pub get`

4. Run: `flutter pub run flutter_launcher_icons`
