// ========== X-VALUE CONSIDERATIONS ==========
// Cards that need Grenzo cast with higher X:

// X=0 (Power 2) - Most of deck works here:
// All power 0-2 creatures flip successfully

// X=1 (Power 3) - Some key cards need this:
// Woe Strider, Midnight Reaper, Grim Haruspex, Ingot Chewer
// Fleshbag Marauder, Plaguecrafter, Mayhem Devil

// X=2+ (Power 4+) - Rarely needed:
// Fanatic of Mogis, Gray Merchant gets better with more devotion

// RECOMMENDATION: Default to X=0 or X=1
// X=0: Faster, more mana for activations
// X=1: Hits more cards, still affordable at 3 mana

// ========== CUTS TO CONSIDER ==========
// If you need to make room:

LOWEST IMPACT (cut first):
- Burglar Rat (1 card from each opponent is minor)
- Phyrexian Ghoul (redundant sac outlet)
- Cinder Barrens (too many taplands)
- Read the Bones (deck already draws well)

MEDIUM IMPACT:
- Faceless Butcher (risky if bounced/killed)
- Defile (sometimes too conditional)

HIGHEST IMPACT (never cut):
- Viscera Seer (scry puts cards on BOTTOM for Grenzo!)
- Blood Artist / Zulaport Cutthroat (win conditions)
- Reito Lantern (directly manipulates bottom of library)
- Ashnod's Altar (infinite mana potential)
